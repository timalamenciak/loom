# Glossary

**Active schema**
: The CAMO schema version selected for a project. It determines the annotation
  form's fields, choices, and requirements.

**Assignment**
: A specific article allocated to an annotator for work and status tracking.

**CAMO**
: Causal Mosaic, the schema Loom uses to represent causal annotations.

**Canonical text**
: Server-extracted article text used for stable character offsets and excerpt
  selection.

**Causal edge**
: A directed causal relationship from a subject node to an object node.

**Complete**
: The status of an edge that an annotator has checked and marked done.

**Draft**
: The initial status of a saved edge that has not yet been marked complete.

**ELMO node**
: A structured description of one side of a claim, including an entity or
  process, an attribute or variable, and a direction or state when specified.

**Excerpt**
: A passage selected from canonical text to ground a node or edge in its source.

**Gold**
: The final accepted status for an edge after adjudication.

**Graph**
: The article-level collection of source context, nodes, edges, provenance, and
  version information.

**Grounding**
: Linking an annotation to the exact source passage that supports it.

**Node**
: One structured side of a causal relationship, used as an edge's subject or
  object.

**Ontology**
: A controlled collection of concepts and identifiers used to describe
  entities and attributes consistently.

**Ontology snapshot**
: The pinned version of the ontology data used by a project or graph, retained
  for reproducibility.

**Returned**
: An assignment status indicating that a reviewer reopened submitted work for
  revision.

**Source doc**
: Article-level bibliographic and study context recorded once and shared across
  the article's annotation.

**Submitted**
: An assignment status indicating that the annotator sent the work for review.
  Submitted work is read-only unless returned.

**Subject and object**
: The cause and effect nodes of an edge, respectively.
