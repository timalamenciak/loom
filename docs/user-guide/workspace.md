# Use the annotation workspace

The workspace keeps the source article and its annotation graph together.

## Session bar

The bar at the top contains:

- the project and article title;
- your active annotation time;
- **Pause timer** or **Resume**;
- **Export YAML**, when a graph exists; and
- **Submit for review**, while the assignment is editable.

The timer counts active work, not simply how long the tab is open. It pauses
when the tab is hidden, after 90 seconds without activity, or when you select
**Pause timer**. Keyboard, mouse, scrolling, or text-selection activity resumes
the idle timer.

## Document pane

The left pane displays the source document.

For a PDF-backed article, you may see:

- **Text**: selectable canonical text used to create excerpts;
- **Markdown**: a layout-aware reading view, when available; and
- **PDF**: the original uploaded document.

Use **Text** when collecting excerpts. Use **PDF** or **Markdown** to check
tables, headings, page layout, symbols, or reading order. **Open PDF** opens the
uploaded file in a separate browser tab.

## Excerpt bin

The excerpt bin holds passages selected from the Text view.

To add an excerpt:

1. Select the relevant text in the document pane.
2. Select **Add to excerpt bin**.
3. Confirm that the passage appears in the bin.

Select one or more excerpt checkboxes, then choose **Use for node** or **Use for
edge**. The chosen passages appear under **Grounding excerpts** in the form.

You can delete an unused excerpt from the bin. Before deleting, confirm that it
is not the evidence you intended to use for another annotation.

## Graph panel

The graph panel lists the article's nodes and edges. Use it to:

- select **+ Add node** or **+ Add edge**;
- open **Source doc**;
- edit an existing node or edge;
- inspect edge status; and
- mark a draft edge done.

## Form panel

The form panel displays the item you are creating or editing. Its fields are
generated from the project's active schema. Required fields are marked, and
validation messages appear beside fields that need attention.

The divider between the article and sidebar can be dragged to resize the
workspace. Loom remembers the sidebar width in your browser.
