# Record article context

**Source doc** stores study and bibliographic information that applies across
the article. Complete it once per article instead of repeating the same context
for every edge.

## Open the form

1. In the graph panel, select **Source doc**.
2. Review the bibliographic values pre-filled from the document record.
3. Add the study-level context supported by the article.
4. Select **Save source document**.

The active schema determines the exact fields. They may include article
section, study location, coordinates, time period, or ecological context.

## Record only shared context

Use **Source doc** for information that genuinely applies to the article or
study as a whole. If a location, period, or condition applies to only one
claim, record it in the relevant edge field when the schema provides one.

## Check pre-filled information

Pre-filled information is a starting point, not a guarantee. Compare titles,
identifiers, and other bibliographic values with the article before saving.

If coordinate lookup is configured, Loom can use entered coordinates to help
populate location fields. Check the result against the article. If lookup is
not configured, enter the supported location information manually.

!!! tip
    Save the source document before creating many edges. This makes the shared
    study context available throughout the article annotation.
