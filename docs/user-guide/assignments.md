# Receive an assignment

Your queue contains the articles assigned to you across Loom projects.

## Open your queue

1. Log in to Loom.
2. Select **My queue** in the main navigation.
3. Find the article you want to work on.

Each row identifies the article, project, available text, assignment status,
and assignment date.

## Understand the text label

| Label | Meaning | What to do |
| --- | --- | --- |
| **PDF** | A PDF and extracted text are available | Use the PDF to check layout and the Text view to collect excerpts |
| **Abstract** | Only abstract text is available | Annotate only claims supported by the abstract |
| **None** | No annotatable text is available | Ask a project administrator to attach a PDF or add text |

Do not reconstruct missing article content from memory or another source.

## Understand assignment status

| Status | Meaning |
| --- | --- |
| **Assigned** | You have not opened the annotation workspace yet |
| **In progress** | You have opened the workspace and can edit the annotation |
| **Submitted** | You sent the annotation for review; it is read-only |
| **Reviewed** | A reviewer has processed the submitted work |
| **Returned** | A reviewer returned the assignment for more work |

## Start annotating

1. Select **Open** beside the article.
2. Review its title, authors, abstract, and text availability.
3. Select **Start annotating**.

Opening an **Assigned** or **Returned** article advances it to **In progress**.
Loom also starts an active work session.

!!! warning "Work only on your own assignment"
    Project membership does not make every article editable. Loom requires an
    assignment for the current article and annotator.
