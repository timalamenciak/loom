# Release Prompt Template

Objective:

Prepare this project for release:

[version / milestone]

Relevant context:

- Read `.ecoweaver/publication.md`.
- Read `.ecoweaver/review.md`.
- Read `.ecoweaver/testing.md`.
- Inspect changelog, README, packaging files, schemas, and tests.

Constraints:

- Do not make broad feature changes.
- Focus on correctness, documentation, packaging, and reproducibility.
- Preserve compatibility notes.
- Report blockers rather than hiding them.

Checklist:

- Version is updated consistently.
- Changelog/release notes are updated.
- README installation and usage still work.
- Tests pass or failures are documented.
- Schema artifacts are current where applicable.
- License and citation metadata are present.
- JOSS/research software expectations are considered.

Final response:

Provide a release-readiness summary, changed files, checks run, and blockers.
