# Prime Project Context

Read the project context before making changes.

1. Read `AGENTS.md`.
2. Read `.ecoweaver/index.md`.
3. Inspect the repository structure.
4. Read the README and relevant docs.
5. Identify the likely subsystem for the task.
6. Summarize:
   - project purpose
   - relevant files
   - likely commands
   - risks or uncertainties

Do not edit files during priming.
