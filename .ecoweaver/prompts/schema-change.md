# LinkML Schema Change Workflow

1. Read `.ecoweaver/linkml.md`.
2. Inspect the authoritative schema file.
3. Search for existing classes, slots, enums, and mixins.
4. Determine whether the change is additive or breaking.
5. Update source schema.
6. Regenerate generated artifacts.
7. Update examples, docs, and tests.
8. Run schema validation.
9. Report migration implications.
